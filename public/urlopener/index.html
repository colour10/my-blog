初次使用本站，请务必注意：

■ 您的浏览器必须打开 Javascript 支持（所有浏览器均默认支持，如果您手动关闭了请将其打开）！

■ 为了良好的打开体验，请尽量使用支持多标签的浏览器，推荐使用 火狐浏览器 或 Chrome！

■ 大部分浏览器都对弹出窗口的行为和次数进行了限制，请参考 常见问题 相应的教程对你的浏览器进行一番设置，不然无法正常使用！

■ 如果按教程设置后依然无法正常使用，请按 Ctrl + F5 强制刷新浏览器！

■ 网址的长度不受限制，鼠标双击输入框可选中全部的网址！

经测试，同时打开30个或以上的网址会影响浏览器和电脑运作。继续打开！
